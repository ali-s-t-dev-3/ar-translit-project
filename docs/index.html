    How it works

    The original Python backend uses longest-match transliteration rules, a small Arabic dictionary, frequency ranking, and a bigram map for next-word prediction.

    What changed for the web

    The engine was ported into browser JavaScript so GitHub Pages can host it with no server, database, login, or paid infrastructure.
